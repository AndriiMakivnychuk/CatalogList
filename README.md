CatalogList manager:/n
To run an application in the terminal type "docker compose up -d"/n
To check swagger move to "http://localhost:3000/api/docs"

List of endpoints(for more datails go to swagger):/n
GET catalogs - get all catalogs/n
GET catalogs:id - get a catalog by id/n
POST catalogs - to create a catalog/n
PATCH catalogs:id - to change/update catalog/n
DELETE catalogs:id - to delete a single catalog/n
DELETE catalogs/bulk-delete - delete a few catalogs from the list in a bulk action/n
